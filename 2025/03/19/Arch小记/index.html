Arch小记：Vmware手动安装ArchLinux

前言：根据网上各博客以及wiki才成功安装，如有错误，期待指正！

1.前置

Vmware官网和Arch官网，在官网上下载即可
这是Vmware的一个官方仓库。
然后去Vmware里面创建虚拟机，创建完后，打开虚拟机目录，修改对应的.vmx文件，在开头添加一下代码：
firmware = "efi"
然后打开虚拟机，进去后会有一个选项，直接第一个回车即可

虚拟机硬盘大小建议40G以上

2.安装前准备

2.1调整控制台字体

setfont ter-132b

2.2验证启动模式

查看UEFI位数，返回64或32表示用UEFI模式启动：
cat /sys/firmware/efi/fw_platform_size
64

2.3设置时区

timedatectl set-timezone Asia/Shanghai
timedatectl
看到Asia/Shanghai说明成功

2.4分区

使用fdisk -l命令查看分区，并用cfdisk进行分区，这里我的磁盘名字为/dev/sda，具体按照实际磁盘名为准
fdisk -l
cfdisk /dev/sda
为EFI系统分区（启动系统的引导）创建1G的空间，为交换分区创建4G的空间，剩余给根分区

我分配的空间大小是建议大小，可以根据实际情况自主分配

EFI系统分区的类型（type）为efi system，交换分区为swap，系统文件类型不用改，分完区后记得写入（write），它会提示你是否要写入，键入yes以完成分区。（每一个分区都要write！并且必须输入yes否则不会写入）
最后再用fdisk -l查看分区

之后用efi_system_partition、swap_partition和root_partition表示EFI系统分区、交换分区和根分区

2.5格式化分区

这里我用btrfs的文件系统，具体文件系统的差别自行查询

格式化EFI系统分区：
mkfs.fat -F 32 /dev/efi_system_partition
初始化交换分区：
mkswap /dev/swap_partition
格式化根分区：
mkfs.btrfs /dev/root_partition -f
建立btrfs子卷：
mount /dev/root_partition /mnt
btrfs subvolume create /mnt/@
btrfs subvolume create /mnt/@home
umount /dev/root_partition

2.6挂载

我们要进入系统安装软件包，所以要先挂载
挂载根分区和EFI系统分区：
mount /dev/root_partition /mnt -o subvol=@
mount /dev/root_partition /mnt/home -o subvol=@home --mkdir
mount /dev/efi_system_partition /mnt/boot/efi --mkdir

这里一定要先挂载根分区，再挂载EFI系统分区，最后挂载其他分区。否则肯可能遇到无法启动系统的情况（摘自wiki）

启用swap分区：
swapon /dev/sda2

3.开始安装！

3.1更改镜像源

这里采用Wiki中的方法，全部换成国内的镜像源：
curl -L 'https://archlinux.org/mirrorlist/?country=CN&protocol=https' -o /etc/pacman.d/mirrorlist
换完之后，用文本编辑器将镜像源前的注释删掉：（将#删除）
vim /etc/pacman.d/mirrorlist

3.2安装必要软件

安装一些必要工具：
pacstrap -K /mnt base base-devel linux linux-firmware linux-headers git grub efibootmgr networkmanager dhcpcd neovim btrfs-progs

grub和efibootmgr是制作引导时需要的工具
btrfs-progs为文件系统所需工具

4.配置系统

4.1前置设置

生成fstab文件：
genfstab -U /mnt > /mnt/etc/fstab
chroot到新系统：
arch-chroot /mnt

4.2时区与语言设置

设置时区：
ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
hwclock --systohc
生成locale信息：
locale-gen
删除en_US.UTF-8 UTF-8和zh_CN.UTF-8前的注释：
nvim /etc/locale.gen
创建locale.conf文件：
nvim /etc/locale.conf
在这个文件中写入LANG=en_US.UTF-8

4.3更改pacman配置

取消Color和ParallelDownloads前的注释
nvim /etc/pacman.conf
设置吃豆人彩蛋：
ILoveCandy
然后更新：
pacman -Syy

4.4设置密码和添加用户

设置主机名：
nvim /etc/hostname
第一行写上你的主机名
设置root密码：
passwd
添加用户并设置密码：
useradd -m -G wheel <username>
passwd <username>
设置用户sudo权限：
nvim /etc/sudoers
删除%wheel ALL=(ALL:ALL) ALL前的注释

nvim可以使用wq!强制保存退出

你可以用su命令进入用户：
su <username>
ctrl+D退出登录

4.5其他设置

启用网络服务：
systemctl enable dhcpcd
systemctl enable NetworkManager
添加钩子：
mkinitcpio -P

4.6安装引导：

相关软件包在3.2中已经安装过了，这里直接设置引导：
grub-install --target=x86_64-efi --efi-directory=/boot/efi --bootloader-id=ARCH
grub-mkconfig -o /boot/grub/grub.cfg

END

退出chroot，取消挂载并关闭系统
exit
umount -R /mnt
shutdown -h now
最后别忘了在虚拟机设置中取消ISO，然后再开虚拟机，看看能不能进入系统吧
